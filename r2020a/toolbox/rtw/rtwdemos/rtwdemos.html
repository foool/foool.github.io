Simulink Coder and Embedded Coder Demonstrations

This model presents several features and capabilities of Simulink Coder and Embedded Coder. The model is organized into subsystems by these catagories:

  • Modeling
  • Multirate
  • Fixed-point
  • Accelerated Simulations
  • Optimization
  • Function Packaging
  • File Packaging
  • Data Packaging
  • Target Environment
  • Custom Code
  • Interfaces
  • Code Verification

Once you open this model, double-click on a topic of interest. You will be presented a series of individual demonstrations under that topic.
